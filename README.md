# DNA-Sequence-Manipulator
 A Python-based DNA sequence analysis toolkit that allows users to perform various operations on DNA sequences, including calculating length, counting nucleotides, finding patterns, and determining GC content.
## Features

- Calculate DNA sequence length.
- Count nucleotides (A, T, C, G) in the sequence.
- Find patterns within the DNA sequence.
- Calculate GC content as a percentage.

## Usage

To use this toolkit, clone the repository and run the provided Python scripts.

## Installation

1. Clone the repository to your local machine.
2. Install Python and any required dependencies.
3. Run the Python scripts to analyze DNA sequences.
